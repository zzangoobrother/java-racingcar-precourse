# 자동차 경주 게임
## 진행 방법
* 숫자 야구 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.

## 과제 제출 과정
* [과제 제출 방법](https://github.com/next-step/nextstep-docs/tree/master/precourse)

## 구현할 기능 목록
1. 경주에 참여할 n대의 자동차 이름을 입력받는다.
* 1대 이상의 자동차가 존재한다.
* 자동차 이름에 공백이 오면 안된다.
* 자동차 이름은 1자이상 5자 이하 이다.
* 중복된 자동차 이름은 존재 안한다.

2. 사용자에게 시도할 횟수 입력받는다.
* 1이상의 수가 입력되어야 한다.
* 정수가 들어와야 한다.

3. 주어진 회수 만큼 이동한다.
* 0~9 사이의 랜덤 숫자받는다.
* 4이상이면 전진, 3이하이면 정지한다.

4. 게임 종료 후 우승자를 출력한다.
* 한명 이상의 우승자를 출력한다.(동점자 가능)
